KAZI V19 CONTACT MODAL WHATSAPP

Correction:
- Dans Marketplace, le bouton affiche CONTACT.
- Quand on clique CONTACT:
  * une petite fenêtre s’ouvre.
  * elle montre le service.
  * elle montre le numéro WhatsApp du vendeur.
  * elle donne un lien “Ouvrir WhatsApp du vendeur”.
  * le vendeur reçoit une notification dans son compte.
  * un email est simulé.
  * le compteur contact augmente.
  * les étoiles de popularité montent.

WhatsApp:
- Le vendeur doit mettre son numéro au format international:
  exemple RDC: 243810000000
